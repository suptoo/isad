# DIP Exam — MCQ Practice Lab

Mobile-first Digital Image Processing study hub for **CSE 4106**. Six interactive chapters, 500 MCQs with instant explanations.

## Open locally

1. Clone or download this repo
2. Open **`home.html`** in any browser (Chrome, Edge, Firefox)
3. Pick a chapter and start practicing

No server or install required.

## Design

- **`css/site.css`** — shared header, navigation, home page
- **`css/chapter.css`** — unified quiz UI across all chapters
- **`js/site.js`** — chapter menu, mobile drawer, progress memory

Optimized for phone reading with large tap targets; scales cleanly on desktop.

## How it works

- Tap an answer → **blue** = correct, **red** = incorrect
- Read the explanation, then continue
- Use the ☰ menu on any page to jump between chapters
- No scores — learn at your own pace
